# How to Run the Panamax Trading Game App

## Local Setup
1. Install Python 3.9+
2. Run `pip install streamlit pandas matplotlib`
3. Launch the app: `streamlit run game.py`

## Streamlit Cloud Deployment
1. Push this folder to a GitHub repository
2. Visit https://streamlit.io/cloud and link your repo
3. Select `game.py` as the entry point
4. Ensure `/data/` folder is included
5. App will be live and shareable via link
